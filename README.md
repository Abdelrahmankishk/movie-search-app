# Movie Search App

A React app to search movies using the OMDb API.

## Setup Instructions

1. Clone this repo:
   ```bash
   git clone <your-repo-link>
   ```
2. Install dependencies:
   npm install

3. Add your OMDb API key in Home.jsx and MovieDetails.jsx.

4. Run the app:
   npm start

Features:
Search for movies

    View details

    Styled with Bootstrap
